# DezzOS v1.0  
**TheBrokenDezz | 19.11.2025**

**Хакерская консольная ОС-симулятор, написанная мной! на чистом C++ за 3-4 дня с нуля.**

Никаких библиотек.
Никаких фреймворков.
Только WinAPI, 350 строк и визуал взлома.

## Возможности v1.0
- Полноценная загрузочная анимация (21 кадр)
- Генерация своего IP
- Поиск жертв (IP + MAC)
- Полный взлом по IP+MAC
- Подключение к жертве
- Удаление системы жертвы (y/n)
- Сохранение/загрузка сессии
- Три уровня прав (user → root → hack)
- Пасхалки (пока одна)

## Команды
help, clear, info, quit
sudo root DezzOS on/off/hack
sudo search ip/mac
sudo hack <IP> <MAC>
sudo connect 0 / list
